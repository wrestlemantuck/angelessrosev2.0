## FAQ

### Why is this open source?
I made this open source so people can look through the code themselves. Some people might assume it has malware, so this lets anyone verify what it actually does. Also because I couldnt be bothered to obfuscate the dll.

### Why are the variable names like `thisIsAVar`?
I use camelCase for variables. That’s just the naming style I prefer.

### Why are there so many comments?
The comments are there to help people understand what the code is doing and how to use the methods.

### Why is `Patches/LocalAuthorityPatch.cs` needed?
This patch exposes more functionality inside `CrittersManager`. For example, it allows things like despawning all critters.

### Do I need to give credit if I use methods from here?
Credit isn’t required, but it would be appreciated if you use methods from this project.
